如大家所见，在博客建立之后只有一张图片，整个页面看起来非常的单薄。所以我添加了鼠标样式和鼠标点击特效。同时在本次更新最最重要的是使用了vercel加速访问。

鼠标样式和特效更新

鼠标样式比较简单，直接在theme的css文件夹中新建一个mouse.css文件，然后再在下面粘贴如下代码

body {
    cursor: url(https://cdn.jsdelivr.net/gh/honjun/cdn@1.6/img/cursor/normal.cur),
        default;
}
a,
img {
    cursor: url(https://cdn.jsdelivr.net/gh/honjun/cdn@1.6/img/cursor/ayuda.cur),
        default;
}
p {
    cursor: url(https://cdn.jsdelivr.net/gh/honjun/cdn@1.6/img/cursor/texto.cur),
        default;
}

然后回到theme的layout文件夹中的layout.ejs文件引用这个css即可。

部署到vercel加速访问

github由于众所周知的原因，访问慢而且是不是抽风访问不了。为了解决这个问题我们使用了vercel进行加速访问。

先注册一个 Vercel 账号，通过 Github 注册
注册成功后跳转到你的首页，点击 New project，跳转后在 Import Git Repository 中选择你的 Hexo 博客，跳转后点击 Deploy 上传到 Vercel
上传完成点击 Goto Dashboard 来到项目主页，选择顶部的 Settings，在 Project Name 中更改你网站的名称。

完成后vercel给了我一个https://leolyx54110-github-io.vercel.app/的链接。用这个链接访问可以感觉到比之前的github直连块多了。（PS.其实我什么感觉都没有，毕竟我在互联网上已经是一位常驻香港人了）
